# Default starter for Gridsome

This is the project you get when you run `gridsome create new-project`.

### 1. Install Gridsome CLI tool if you don't have

`npm install --global @gridsome/cli`


### 2. Install Dependencies

`npm install`
or
`yarn install`

### 3. Develop

`gridsome develop`

### 4. Build
`gridsome build`
